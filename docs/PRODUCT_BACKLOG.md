# 摄影布光软件扩展需求（按痛点）

## 1. 灯光与设备管理

- 灯具预设库：Godox/Profoto 常见机型参数模板（功率、造型灯、回电时间）。
- 附件链路：主附件 + 次附件 + 色片叠加，实时计算等效光损。
- 灯组联动：A/B/C 组总开关、比例联动、编组复制粘贴。

## 2. 场景搭建效率

- 吸附系统：设备移动支持地面网格吸附、与模特距离吸附。
- 标尺与量角器：实时显示相机-模特、灯-模特距离和俯仰角。
- 场景资产面板：背景纸、反光板、黑旗白旗、C-Stand 常用预置。

## 3. 渲染与成片交付

- 一键批量出图：同一布光方案下自动输出多个焦段/曝光组合。
- LUT 仅用于导出预览层：实时预览保持物理线性，避免误导布光。
- 输出归档：在本地作品库里保存“图片 + 参数快照 + 灯位 JSON”。

## 4. 协作与生产流程

- 布光方案版本历史：每次修改记录 diff，可回滚。
- 团队共享方案：导出/导入 `.lighting.json`，支持跨设备还原。
- 助理执行单：自动生成“器材清单 + 摆位图 + 参数表”。

## 5. 技术稳定性

- 大模型加载异步队列：显示进度、失败重试、自动降级占位模型。
- 渲染性能分级：编辑态（低开销）/导出态（高质量）双管线固定。
- 自动错误边界与恢复：避免白屏，提供“恢复默认场景”入口。
